# 📚 PyCiteMap

PyCiteMap 是一个基于 Python 的 **自动化文献计量分析工具**，能够从公开学术数据库中自动检索相关文献，并生成多种文献计量可视化结果。

用户只需输入研究关键词，系统即可自动完成：

- 文献检索
- 元数据清洗
- 关键词提取
- 文献计量分析
- 可视化展示

项目旨在提供一个 **轻量级、自动化、可复现的文献计量分析工具**，作为 CiteSpace、VOSviewer 等传统文献计量软件的补充。

---

# ✨ 项目功能

### 🔎 自动文献检索
根据用户输入关键词，从公开学术数据库自动获取文献元数据。

### 🧹 数据清洗与标准化
对检索得到的文献数据进行处理，包括：

- 去重
- 字段标准化
- 缺失值处理
- 关键词提取

### 📊 文献计量分析
自动构建文献计量指标，包括：

- 年度发文趋势
- 高频关键词统计
- 关键词共现关系

### 📈 可视化分析
系统可以生成以下可视化结果：

- 发文趋势图
- 关键词词云图
- 关键词共现网络
- 关键词聚类网络

### 🌐 Web 可视化界面
项目提供基于 **Streamlit** 的 Web 交互界面，用户可以直接在浏览器中进行分析。

---

# 🧠 系统架构

PyCiteMap 采用模块化设计，整体流程如下：

关键词输入
↓
公开数据库文献检索
↓
元数据清洗与标准化
↓
关键词提取
↓
文献计量分析
↓
可视化输出

# 📊 可视化结果示例

系统可自动生成以下文献计量图谱：

### 发文趋势图
展示某一研究领域的年度发文变化趋势。

### 关键词词云图
展示文献中出现频率较高的关键词。

# ⚙️ 安装方法

## 1 克隆项目

```bash
git clone https://github.com/ElysiaDeFountaine/PyCiteMap.git
cd PyCiteMap
````

---

## 2 安装依赖

```bash
pip install -r requirements.txt
```

---

# 🚀 使用方法

## 命令行版本

直接通过命令行运行文献计量分析：

```bash
python app.py --query "companion ai elderly care" --max-results 200
```

运行完成后，生成的可视化结果将保存在：

```
outputs/
```

---

## Web 可视化界面

启动 Web 应用：

```bash
streamlit run web_app.py
```

浏览器打开：

```
http://localhost:8501
```

输入研究关键词并点击 **开始分析** 即可自动生成文献计量图谱。
